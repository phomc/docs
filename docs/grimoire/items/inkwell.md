# Inkwell
Inkwell is an item used to create Unidentified Grimoire. See: Scriptorium

Ingredients:
- x1 Glass Bottle
- x2 Gemstone
- x6 Ink Sac

Note:
- Inkwell can not be stacked
- While Musgravite is fire-resistant, its corresponding inkwell is not
